Grinch

A tiny, fast macOS browser router. Set it as your default browser; it routes each URL to the right browser based on rules in a JavaScript config.

Download DMG View on GitHub

macOS 13+ · Universal binary · Signed & notarized

What it is

  • ~1500 lines of Rust + a small embedded JS prelude. No bundler, no transpiler, no Electron.
  • Hot-path resolve in nanoseconds on Apple Silicon. End-to-end click-to-browser latency is in the single-digit-millisecond range, dominated by macOS event dispatch and the target-browser launch — not the engine.
  • ~16 MB resident memory.
  • Native JavaScriptCore. Config is real JavaScript — simple cases look like data, full power available when you need it.
  • Finicky v4 compatible. Most v4 configs work unchanged: finicky.matchHostnames, finicky.notify, ctx.opener.*, the options block, profile shorthand.
  • Menu-bar app. Reload config, test a URL, open the log — without leaving the menu bar.

Install

Grab the DMG, drag Grinch.app to /Applications, then set Grinch as your default browser in System Settings.

DMG=$(curl -fsSL https://api.github.com/repos/jamtur01/grinch/releases/latest \
  | grep -oE '"browser_download_url": "[^"]*\.dmg"' | cut -d'"' -f4)
curl -fsSL "$DMG" -o /tmp/grinch.dmg
hdiutil attach -nobrowse -quiet /tmp/grinch.dmg
ditto "/Volumes/Grinch "*/Grinch.app /Applications/Grinch.app
hdiutil detach "/Volumes/Grinch "* -quiet
open /Applications/Grinch.app

Always install to /Applications directly — running out of ~/Downloads triggers Gatekeeper translocation, which makes the same app appear multiple times in the default-browser picker.

Or build from source:

git clone https://github.com/jamtur01/grinch
cd grinch
make run

Configure

Drop a JavaScript file at ~/.config/grinch.js. First match wins.

module.exports = {
  default: "org.mozilla.firefox",

  rules: [
    // Slack invitations open in Chrome.
    { match: from("com.tinyspeck.slackmacgap"),
      open: "Google Chrome" },

    // GitHub goes to a work Chrome profile.
    { match: finicky.matchHostnames(["github.com", "*.github.com"]),
      open: { name: "Google Chrome", profile: "Work" } },

    // Zoom links open the native client.
    { match: /^https:\/\/[a-z0-9-]+\.zoom\.us\/j\//,
      open: "us.zoom.xos" },
  ],

  rewrite: [
    // Force HTTPS on a few hosts.
    { match: /^http:\/\/(developer\.apple\.com|news\.ycombinator\.com)/,
      url: ({ url }) => ({ ...url, protocol: "https" }) },
  ],
};

Reload after editing: kill -HUP $(pgrep -f Grinch.app/Contents/MacOS/Grinch) — or click Reload Config in the menu.
